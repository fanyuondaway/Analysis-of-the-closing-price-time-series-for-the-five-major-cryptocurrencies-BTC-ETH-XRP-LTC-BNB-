# Analysis-of-the-closing-price-time-series-for-the-five-major-cryptocurrencies-BTC-ETH-XRP-LTC-BNB
This project primarily focuses on loading and conducting simple data analysis on cryptocurrency datasets.
The following content is mainly some data processing and analysis of the crpto-markets data set. Previously, by understanding the processing methods of other friends and giving me a lot of inspiration, 
I finally created a visualization of low currency prices and high buying prices based on the data of some time nodes. The main body of the code was optimized with the help of Qwen and GLM.
